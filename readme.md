Este é o código do jogo "Termo" no terminal em linguagem C. 

Nele, o jogador deve descobrir a palavra certa em 6 tentativas. Para isso cada tentativa, o jogador precisa digitar uma palavra:

Caso alguma letra da palavra apareça colorida de verde, isso significa que a letra está presente na resposta, e que ela está no lugar correto.

Caso alguma letra da palavra apareça colorida de amarelo, isso significa que a letra está presente na resposta, porém, ela está no lugar errado.

Caso alguma letra da palavra apareça colorida de vermelho, isso significa que a letra não está presente na resposta.
